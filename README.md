Simple, QMK one-button keyboard firmware for a Pro Micro ATMega32U4. Defaults to acting as a space bar.

qmkeybox_source   : QMK source files.  
step              : .step files of 3D prints  
stl               : .stl 3D print files for enclosure.  
QMKeybox.hex      : compiled firmware.  
VIA_QMKeybox.json : layout file for remapping in VIA.
